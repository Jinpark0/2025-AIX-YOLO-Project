### 프로젝트 개요
+ 프로젝트명 : 밤 및 악기후(비, 안개, 눈)에서 YOLO를 활용한 객체 인식 성능 향상 연구

+ 목적 : 자율주행 시 여러 악조건에서 YOLOv11을 활용하여 객체를 정확히 인식하기 위함

+ 목표 : 인식하기 쉬운 clear한 daytime에 비해 악조건에서 유의미한 성능 향상

### 1. 제안 (Proposal)

#### 동기 및 배경

자율주행 시에는 카메라가 여러 객체들을 정확하게 인식하는 것이 필요합니다. 이 때 최신 모델인 YOLOv11을 활용하여 사물들을 인식할 수 있습니다. 낮이면서 맑은 날씨에는 성능이 안정적으로 나오지만, 어두운 밤이나 안개, 비, 눈이 영향을 미칠 경우 성능이 크게 떨어지는 경향이 있습니다. 따라서, 이를 보완하기 위해 학습 시 Data Augmentation과 같은 기법을 사용하여 Object detection 성능을 끌어올리기 위한 연구가  필요합니다.

+ 활용 모델
  
  + YOLOv11
